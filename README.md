# bee-shop-b2c-web
一个用Vue写的B2C商城 web端
技术栈： Vue2.0、axios、Webpack、Vant UI、Vuex、Vue Router

## 后端和管理端源代码 
* [go-shop-b2c](https://github.com/CleverBamboo/go-shop-b2c)
* [go-shop-b2c-admin](https://github.com/CleverBamboo/go-shop-b2c-admin)

## 安装依赖
建议使用 yarn，因为项目包含 yarn.lock
```
yarn install
```

### 运行项目
```
yarn serve
```

### 编译项目
```
yarn build
```
